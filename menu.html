CUSTOM VIM KEYBINDINGS:

Mini Map:

Toggle - press 'Enter'

Easy Motion:

Moving back - press ',' twice followed by 'b'
Moving foward - press ',' twice followed by 'w'

Nerd Commenter:

Comment and uncomment code - hold 'SHIFT' and press '+' twice
Toggle JSX commenter mode - press ',' then 'c' then 'a'

Quick Run:

Run a single block of code - press 'F9'
Run all the code on a page - press 'F10'

Nerd Tree:

Make a new file or folder - press 'm' and press 'a'
Move a file or folder - press 'm' and press 'm'
Delete a file or folder - press 'm' and press 'd'
Toggle Nerd Tree - hold 'CTRL' and press 'n'
Move up and down - 'j' and 'k' Toggle Nerd Tree - hold 'CTRL' and press 'n'
Toggle Nerd Tree - hold 'CTRL' and press 'n'
Open a folder - 'F10'
